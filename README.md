# First Spring Boot Project - Task 1

## Description
This is my first Spring Boot project created for Akademia Finansów i Biznesu Vistula.

## What it does
- Created a Spring Boot project using Spring Initializr
- Written first Spring MVC Controller
- Added Thymeleaf template with greeting page
- Displays an image from static resources

## How to run
1. Open the project in IntelliJ
2. Run FirstProjectJavaSpringApplication
3. Open browser and go to http://localhost:8080
4. Go to http://localhost:8080/greeting?name=Vistula

## Technologies used
- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- Maven
- Lombok
